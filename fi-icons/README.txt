Fully Inhabit - Icon Set v0.2
=============================
Construction: single-weight line, 24px grid, charcoal #2c2c2a.
Rule: open ring = node/entity, solid dot = list marker.
Tiles use paper ground #f4ece9. All glyphs are mask-free (clean on any background).

svg/              Scalable transparent glyphs (thin stroke). Site / large use.
png-tile/         512px paper tiles (heavier stroke). Notion page icons. Dark-mode safe.
png-transparent/  512px transparent glyphs (heavier stroke). Light surfaces.

NOTION: Page icon > Custom > Link, paste the raw GitHub URL of the png-tile file.
Keep the lightbulb for AI pages. Repo must stay PUBLIC for raw URLs to resolve.
